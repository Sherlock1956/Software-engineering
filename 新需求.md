- 元数据处理

  - **属主（Owner）和权限（ACLs）**：

    - 使用 `GetFileSecurity` 提取源文件的安全描述符（包含属主和权限）。
    - 使用 `SetFileSecurity` 将安全描述符应用到目标文件。

    **时间戳**：

    - 原有代码中使用 `GetFileTime` 和 `SetFileTime`，可以保留文件的创建时间、访问时间和修改时间。

  - **文件属性**：
    - 使用 `GetFileAttributes` 和 `SetFileAttributes`，确保文件的只读、隐藏等属性被保留。
